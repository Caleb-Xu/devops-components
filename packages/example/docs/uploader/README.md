# Uploader 上传文件

源自度量平台的一款上传组件，通过按钮触发弹窗，在弹窗内提供一个上传入口供用户选择文件上传。

## 示例

```vue
<template>
    <div>
        <Uploader></Uploader>
    </div>
</template>

<script>
import Uploader from "@devops/uploader";

export default {
    components: { Uploader },
    data() {
        return {};
    },
};
</script>
```
